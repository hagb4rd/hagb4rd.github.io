what is Gitter?

man irc.gitter.im (this service is still very much beta)
  1. 1. Click here to login and get your token
  2. 2. Connect to irc.gitter.im with your IRC client using SSL(port 6667 or 6697)
  3. 3. Provide the the token we give you (login to see token) as the Server Password
  4. 4. /NICK your github username
  5. 5. Profit

100% compatible with Lynx. Seriously.
